┌──────────────────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

   Problem: Python not installed
   Solution: Download from https://www.python.org/downloads/
            Install 3.10+ and check "Add to PATH"

   Problem: Node.js not installed
   Solution: Download from https://nodejs.org/
            Install LTS version (18+)

   Problem: Port already in use
   Solution: Run STOP-PROJECT.bat or let START-PROJECT kill it

   Problem: Something not working
   Solution: Run DEBUG-PROJECT.bat and check diagnostic report

   Problem: Need fresh start
   Solution: STOP → Delete backend/negotiator.db → START
